## 📝 Description / 项目简介
A systematic journey of learning Artificial Intelligence and Neural Networks from scratch using C++. This repository records my progress from basic perceptrons to advanced deep learning architectures.
这是一个使用 C++ 从零开始系统学习人工智能和神经网络的项目记录。本仓库记录了从基础感知器到高级深度学习架构的进化过程。

---

## 🚀 Roadmap / 学习路线图

### Phase 1: Fundamentals (Current)
- [x] **01_Perceptron**: Implementation of a single-layer neuron to solve linearly separable problems (e.g., AND/OR gates).
  - *Key Concepts*: Weights, Bias, Step Function, Supervised Learning.
- [ ] **02_Linear_Separability**: Exploring the limitations of Perceptrons (The XOR Problem).

### 第一阶段：基础核心 (进行中)
- [x] **01_Perceptron**: 实现单层神经元，解决线性可分问题（如 AND/OR 逻辑门）。
  - *核心概念*: 权重、偏置、阶跃函数、监督学习。
- [ ] **02_Linear_Separability**: 探索感知器的局限性（异或 XOR 问题）。
